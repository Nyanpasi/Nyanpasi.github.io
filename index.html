22. 返回URL Return URL

通常, 返回url的时候最好是返回绝对url, 而不是返回相对url.

这样做的优势在于：

  1. 更明确
  2. 减轻客户端的负担
  3. 当在诸如JSON之类的结果中找不到原始URL时, 也不会产生歧义
  4. 很容易地使用带有超链接的html样式
